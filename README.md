# Web Eye Morse

目のウインクでモールス信号を入力し、文字を打つWebアプリです。

🌐 **[Webアプリを開く](https://taikiishii.github.io/web-eye-morse/)**

## 概要

カメラで顔を認識し、目の開閉（EAR: Eye Aspect Ratio）を検出することでモールス信号を入力します。キーボードや手が使えない状況でも文字入力が可能です。

## 操作方法

| 操作 | 入力 |
|------|------|
| 右目を0.5秒以上閉じる | ・（ドット） |
| 左目を0.5秒以上閉じる | －（ダッシュ） |
| 両目を1秒以上閉じる | シーケンス消去 / 一文字削除 |
| 2秒間入力なし | 文字を確定して出力 |

## 機能

- **アルファベットモード** — 英字・数字・記号をモールス信号で入力
- **ひらがなモード** — JARL標準の和文モールス符号でひらがなを入力
- **EAR閾値調整** — 目の閉じ具合の判定感度をスライダーで調整
- **文字確定タイミング調整** — 待機時間をスライダーで調整（1〜4秒）
- **テキストエリア** — 入力した文字をそのままコピーして会話に使用可能

## 技術

- [MediaPipe Face Mesh](https://google.github.io/mediapipe/solutions/face_mesh.html) — 顔のランドマーク検出
- 純粋なHTML/CSS/JavaScript（サーバー不要）
- カメラ映像はブラウザ内のみで処理され、外部に送信されません

## ブラウザ要件

カメラへのアクセスが必要です。Chrome / Edge 推奨。
